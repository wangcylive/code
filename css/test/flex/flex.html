flex-shrink: 收缩比例，默认值1；空间不足按设置的比例和基础宽度收缩，如果未设置 flex-basis 按照 width 计算；计算方式如:
shrink-total * (A-flex-basis * A-flex-shrink / A-flex-basis * A-flex-shrink + B-flex-basis * b-flex-shrink)
