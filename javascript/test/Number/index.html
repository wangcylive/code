Number(string)从字符串string非空字符开始转换得到数字，string必须为合法数字串，否则返回NaN

"0.1"返回0.1，".123"返回0.123

"1.1.1"，"134a"返回NaN

""、" "、false、null返回0

true返回1

undefined返回NaN

如果string是对象，则会先调用对象的valueOf()如无则再调用toString()的值来处理

parseInt(string[,radix])从字符串string非空字符开始转换得到整数，遇到小数点或者其他0-9以外的字符就停止，string去掉开头空格以-[0-9]或者[0-9]开头返回数字，否则返回NaN。radix一个整数值，指定转换中采用的基数

"-11vs"返回-11，"23jorden23"返回23，"123.342aaa" 返回123

".123"返回NaN

""、" "、Boolean、null、undefined返回NaN

如果string是对象，则会调用对象的toString()值来处理

parseFloat(string)基本和parseInt()相似，不同的是parseFloat()保留小数点后面数字，以小数点开头的返回小于1的数字

".123"返回0.123

"11.22.aa.bb"返回11.22
